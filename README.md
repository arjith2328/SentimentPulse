## 📌 Project Overview
SentimentPulse is an end-to-end Social Media Sentiment Analysis 
and Trend Intelligence platform built on 1.6 Million tweets from 
the Sentiment140 dataset. It combines NLP, Machine Learning, 
Deep Learning and Reinforcement Learning to analyze public 
sentiment and predict sentiment shifts in real time.

---

## ✨ Key Features

| Module | Description |
|---|---|
| 📊 Sentiment Overview | KPI cards, pie chart, sentiment trend over time |
| 🤖 Model Comparison | VADER vs TF-IDF vs TextBlob accuracy comparison |
| 📈 Trend Intelligence | LDA Topic Modeling + trending keywords word cloud |
| 🎮 RL Agent Alerts | Q-Learning agent predicting sentiment shifts |
| 🗄️ SQL Insights | Complex SQL query results from data warehouse |
